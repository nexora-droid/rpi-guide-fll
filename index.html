The ArchaeoVision Setup Guide

A complete beginner friendly guide to build your own ArchaeoVision!

Before you begin

This guide assumes you have no knowledge of booting up a Raspberry Pi. If this is the case, follow along carefully. If you already have booted (and flashed) your RPI please skip to section 1.

2

Downloading and preparing the operating system

Step 2.1: Download the Raspberry Pi Imager

1. Visit the Raspberry Pi website: https://www.raspberrypi.com/software/

2. Download the Raspberry Pi Imager for your operating system (Windows, macOS, or Linux)

3. Install the Raspberry Pi Imager to your computer

Step 2.2: Preparing the MicroSD card

1. Insert the microSD card into your laptop. Use an adaptor if necessary.

2. Open Raspberry Pi Imager application on your computer

3. Click Choose Device and select Raspberry Pi 5

4. Click Choose OS and select your preferred operating system:

  • Raspberry Pi OS (64-bit) - Recommended for beginners
  • Other options available: Ubuntu, other Linux distributions

5. Click Storage and select your microSD card.

6. Click the gear icon or Edit Settings to configure advanced options such as:

  • Set hostname
  • Enable SSH (for remote access)
  • Set username and password
  • Configure your WiFi credentials like SSID and password
  • Set local settings like timezone and keyboard layout

7. Click Save to save your settings.

8. Click Write to write your configurations to the microSD card.

9. Wait for the process to finish. This ranges from 2 minutes to 10 minutes or more depending on your system.

10. Once the process has finished, safely eject your microSD card from the laptop. (Right click microSD card in file man

3

Physical Setup and Assembly

Inserting the MicroSD card

Insert the prepared microSD card into the microSD card slot on the underside of the Raspberry Pi 5 board. Push it in gently until it clicks into place.

Connecting the display

Connect your monitor or TV to the Raspberry Pi using a micro HDMI cable. The Pi 5 has two micro HDMI ports; use the port labeled HDMI0 (the one closest to the USB-C power port) for the primary display.

Connecting the keyboard and mouse

Plug your USB keyboard and mouse into any of the USB ports on the Raspberry Pi 5. The Pi 5 has two USB 3.0 ports and two USB 2.0 ports.

Connecting the network

If you prefer a wired connection, connect an Ethernet cable to the Ethernet port on the Raspberry Pi. If you configured Wi-Fi in the Imager settings, you can skip this step and connect wirelessly.

Install Cooling

If you have a case with cooling or an active cooler, install it now according to the manufacturer's instructions. The Raspberry Pi 5 can run hot under load, so cooling is highly recommended.

4

Power on and first boot!

Connect the power supply

This is the final step! Once you connect the power, your Raspberry Pi will start automatically.

1. Make sure your monitor is turned on and set to the correct input

2. Connect the USB-C power supply to the power port on the Raspberry Pi 5

3. Plug the power supply into a wall outlet

4. The Raspberry Pi will immediately begin to boot (no power button to press!)

What to expect?

During the first boot, you'll see:

  • A red LED light on the board (indicating power)
  • A green LED that blinks (indicating disk activity)
  • Boot messages scrolling on your screen (colored Raspberry Pi logo may appear)
  • The boot process takes 30-60 seconds on first startup

Initial Setup Wizard

If you didn't configure settings in the Imager, you'll be greeted with a setup wizard that will guide you through:

  • Setting your country, language and timezone
  • Creating a username and password
  • Connecting to WiFi
  • Updating software

In many scenarios, some settings are left out during the Imager setup, so it is best to follow all on-screen instructions given to ensure that no settings are left out and everything operates smoothly. After this you will be ready to use your Raspberry Pi 5. If you face any issues, feel free to open the panel in the corner!

Building ArchaeoVision!!

Materials needed

Raspberry Pi AI Camera

Raspberry Pi 5 (which we setup earlier)

Stable internet connection

1

Connect the Camera onto the Raspberry Pi

1. Power off your Raspberry Pi 5 for safety.

2. Locate the camera connector labeled CAMERA on the board.

3. Gently pull up the plastic clip on the camera connector

4. Insert the ribbon cable from the IMX500 (aka the Pi Camera) camera with the blue side facing the Ethernet port.

5. Push the plastic clip back down to secure the cable.

6. Power on your Raspberry Pi.

2

Update your system

Open the Terminal (on your Raspberry Pi) and update your system packages:

sudo apt upgrade

sudo apt full-upgrade

Running both of these commands in succession updates all packages installed on the system and is a good way to ensure every library is up to date

3

Create a Project Folder and Virtual Environment

Create a folder on Desktop and set up a Python Virtual Environment

# Navigate to Desktop and make a folder called 'arch-vision' or 'archaeo-vision'

cd ~/Desktop

mdkir archaeo-vision

# Navigate into the folder

cd archaeo-vision

# Create a virtual environment with system packages

python3 -m venv --system-site-packages archaeo-vision

# Activate the virtual environment

source archaeo-vision/bin/activate

What to expect:

You should see the name of the venv in the brackets of your terminal prompt, like (archeo-vision)

4

Install Moondream and dependencies

Install the Moondream package and required dependencies (make sure your virtual environment is activated):

# Install Moondream)

pip3 install moondream

# Install numpy version

pip3 install numpy

# Download the int8 quantized model

wget https://huggingface.co/vikhyatk/moondream2/resolve/main/moondream-2b-int8.mf

5

Create the object description script

Create a python script to capture images and describe objects in real-time:

# Create the script file

nano archvision.py

# Add the following code:

from picamera2 import Picamera2

from PIL import Image

import moondream as md

# Initialize the Picamera2

picam2 = Picamera2()

picam2.configure(picam2.create_still_configuration(main={"size": (512,512), "format": "RGB888"}))

picam2.start()

# Load the moondream model

model = md.vl(model="./moondream-2b-int8.mf")

print("Moondream AI Vision System Ready!")

print("Starting continuous object detection...")

print("Press Ctrl+C to exit\n")

while True:

# Capture image as numpy array

image_array = picam2.capture_array()

# Convert numpy array to PIL Image

image = Image.fromarray(image_array)

print("Photo captured. Starting query...")

encoded_image = model.encode_image(image)

# Ask questions about the image

answer = model.query(encoded_image, "Is there a man with a blue jacket in this image?")["answer"]

print(answer)
