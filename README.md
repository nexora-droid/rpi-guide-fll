# ArchaeoVision Building Guide

## Specs
ArchaeoVision is an AI-powered computer vision project built using a Raspberry Pi and camera module to assist in archaeological exploration. 

It captures images in real time and uses machine learning to analyze and identify whether objects in the ground may be potential artefacts.

The current image taking speed is around 2 images every second. It is not very much but adds up in the long time.

Processing speed on inital run (first boot) is around 45seconds, but it becomes faster the more you use it.

Project built for First Lego Leage - Unearthed Singapore 2026.

## What is this guide for?
This guide is for anyone who wants to build their own version of ArchaeoVision

It takes you through the entire process step-by-step, starting from setting up the Raspberry Pi, all the way to running the full AI system.

It also includes an AI chatbot powered by Qwen, which can help you troubleshoot problems and fix errors while building the project.

## Features
1. AI Chatbot via [Hackclub AI](https://ai.hackclub.com/). (Model:qwen/qwen3-32b).
2. Flask Backend, Tailwind CSS + HTML based frontend
3. AI handling via JS.
