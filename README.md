# ArchaeoVision Building Guide

## What is ArchaeoVision?
ArchaeoVision is an AI-powered computer vision project built using a Raspberry Pi and camera module to assist in archaeological exploration. It captures images in real time and uses machine learning to analyze and identify whether objects in the ground may be potential artefacts.

Designed with accessibility in mind, ArchaeoVision is beginner-friendly and can be built using affordable, off-the-shelf components. This project demonstrates how edge AI can be applied in real-world scenarios, bringing intelligent object detection directly into the field without the need for powerful remote servers.

## What is this guide for?
This guide serves as a portable, AI-integrated system to help anyone with the components and interest of replicating ArchaeoVision. It has a detailed guide that has all steps in making the project from start to finish, starting from booting and flashing the Pi up untik the code and nitty gritties. 
This guide also includes a AI chatbot for troubleshooting which runs on Qwen's AI LLM through Hack Club's [AI Program](https://ai.hackclub.com/).
